    \raggedright{For my Computer Hardware class’ optional project, I built a compass that can be programmed to point to any location. Using a STM32NUCLEO-F446ZE microcontroller.\\[3.75pt]
    \begin{minipage}[t]{\linewidth}
        \begin{itemize} [nosep,after=\strut, leftmargin=1em, itemsep=3pt]
            \item Used a GY-NEO6MV GPS module to receive and parse GPS data using USART communication and Direct Memory Access (DMA).
            \item Communicated via I2C with a HMC5883L magnetonomer to read magnet field data to determine my compass’ orientation
            \item Used Pulse-Width-Modulation (PWM) to display the direction and distance to the programmed location on a NEOPixel Ring.
        \end{itemize}
        \end{minipage}
